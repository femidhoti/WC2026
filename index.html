Step 1 of 2 — Free Database Setup

You need a free Firebase account to store shared picks. This takes about 5 minutes and you only do it once.

  1. Go to console.firebase.google.com and sign in with Google
  2. Click Add project → give it any name (e.g. "WC2026Family") → Continue → Create project
  3. In the left sidebar: Build → Realtime Database → Create database → pick any location → choose Start in test mode → set expiry to 2026-08-31 → Enable
  4. Click the ⚙️ gear icon (top-left) → Project settings → scroll to "Your apps" → click the </> Web icon → Register app (any nickname) → you'll see a firebaseConfig block — copy it
